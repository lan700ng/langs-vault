## Tema

- **Default** Tema bawaan Obsidian.
- **[[Minimal]]** Tema Minimalis punya opsi banyak.
- **AnuPpuccin** Tema Minimalis dengan opsi variasi, Catppuccin versi lengkap.
- **FastPpuccin** Tema Minimalis dan versi ringan dari AnuPpuccin.
- **Lagom** Tema Minimalis namun tampilan kurang nyaman, CSS nya cocok di jadikan referensi.
- **Solarized** Tampilan yang cukup nyaman untuk codeblock.

## Snippets

- **UI Tweaks** — UI yang di modifkasi supaya pas dan nyaman di perangkat mobile (versi saya).
- **Number Header** — nomor di setiap header untuk menandakan ukuran header.
- *~~**Spoiler**~~*  — fitur tambahan spoiler
- **Table Tweaks** — render tabel lebih rapi, pas di viewport dengan menambahan #T di kolom tabel.
